Getting Started with HTML

The World Wide Web
The internet functions as a web. It is made up of servers, which host files written in HTML for internet browsers to interpret. When someone wants to go to a website their computer sends an HTTP request to a server, which then sends back the appropriate HTML document for their web browser to interpret.

HTML
Hypertext Markup Language documents contain text content which is the buidling blocks for what the web page will look like when viewed through a browser. To write in HTML one uses a series of different tags that surround elementswhich tell the browser how to display the content. Some of these elements can be described as inlnineor block elements.

When writing code it is important to remember that computers are stupid. Meaning that they are extremely literal and can't make simple corrections that a human could. For example, a small typo could ruin an entire section of code because the computer will only read it exactly how it is written.
